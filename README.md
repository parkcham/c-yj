## ✅ Application UI
###HomeScreen
-디데이 100일 ~ 6000일 표시
-각 디데이 내용 등록 및 수정 가능
-디데이 뷰 스크롤시 애니메이션 적용

###FeedScreen
-게시물을 생성할 때 사진 최대 5개 첨부 가능
-게시물을 수정할 때 사진은 수정할 수 없고 글만 수정 가능
-게시물 이미지 클릭시 원본 변경
-게시물 내용 라인 3줄 제한 ,클릭시 제한 해제

###Calendar
-현재 달 기준 1년 우 스크롤 가능
-달력 스크롤시 현재 달 일경우 Today , 아닐 경우 해당 달 1일 일정 표시
-일정 등록 및 삭제 가능
-일정 뷰 스크롤시 애니메이션 적용
