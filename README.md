## ✅ 주요 기능

### Home
- DDay 100일 ~ 6000일 표시
- DDay 내용 등록 및 수정 가능

##
### Calendar
- 과거 1개월,미래 12개월 일정 등록 및 삭제 가능
- 일정 과거 순으로 자동 정렬

##
### Feed
- 게시물을 생성할 때 사진 최대 5개 첨부 가능
- 게시물을 수정할 때 사진은 수정할 수 없고 글만 수정 가능

- 게시물 이미지 클릭시 resizeMode:contain 
- 게시물 내용 numberOfLines:3 제한 클릭 시 제한 해제
